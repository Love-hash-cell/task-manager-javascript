# 📝 To-Do List Web App

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All / Completed / Pending)
- Local Storage support (data saved even after refresh)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
